Specialized plugins

A plugin is a bridge between OCaml code and some meta-level property of the prover. For instance, a default plugin is bound to the predicate "holds", which states that some (encoded) clause holds in the current problem. The plugin for "holds" provides a view centered on the "holds" predicate, and allows to add "holds" facts and filter on them. Plugins can handle some inner type that can be encoded/decoded to meta-prover-compatible types.
